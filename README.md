# React Single Page E-commerce Website

### **Overview**
This is a modern, responsive **Single Page Application (SPA)** built with **React.js**. The project showcases a complete front-end e-commerce flow, including dynamic product listing, user authentication, and a shopping cart system. It focuses on providing a fast and seamless user experience by utilizing client-side routing and local storage for data persistence.

---

### **Key Modules & Features**

#### **1. Dynamic Product Catalog**
- **Multi-Category Store:** Displays a wide range of products including women's cosmetics, clothing, and electronics.
- **Live Data Fetching:** Product information is dynamically integrated from a structured JSON API.

#### **2. User Authentication & Security**
- **Protected Actions:** Implemented a secure flow where users must be authenticated to perform purchases or access certain features.
- **Access Control:** Ensures a personalized experience and secures user-specific interactions.

#### **3. Advanced Shopping Cart**
- **State Management:** Utilizes React state and local storage to keep track of user selections even after page refreshes.
- **Validation Logic:** Prevents duplicate entries of the same product in the cart, ensuring accurate order quantities.
- **Real-time Updates:** Displays selected items in a clean, tabular format for user review.

#### **4. Responsive UI/UX**
- **Modern Styling:** Built with **Tailwind CSS** for a highly responsive and sleek design.
- **Interactive Elements:** Enhanced with **AOS (Animate On Scroll)** for smooth transitions and an engaging user interface.
- **Seamless Navigation:** Implemented **React Router** for instant page transitions without reloading the browser.

---

### **Technical Stack**
* **Frontend Library:** React.js 
* **Styling:** Tailwind CSS 
* **Routing:** React Router 
* **Data Handling:** JSON API, Local Storage
* **Animation:** AOS Animation 
* **Deployment:** Netlify / Vercel

---

### **How to Run This Project**
1. Clone the repository.
2. Run `npm install` to install dependencies.
3. Run `npm start` to launch the application in your browser.
